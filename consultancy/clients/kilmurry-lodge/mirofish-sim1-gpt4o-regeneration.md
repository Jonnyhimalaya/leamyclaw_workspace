## Part 2 — Key Insights: GPT-4o vs GPT-4o-mini

### What improved on GPT-4o

1. **Structural coherence** — 5 clean thematic sections vs the original's marketing-summary format
2. **Competitor nuance** — Stables Club and Dolans now have distinct reactions (Stables = re-examining strategy, Dolans = supporting + considering collaboration)
3. **Tipping points called out explicitly** — Critical Success Factors section identifies 3 success factors and 3 tipping points
4. **Corporate booker concern finally surfaces** — Section 5 includes the first (thin but present) acknowledgment of corporate event booker worries about student nights affecting their choices
5. **Long-term identity framing** — Section 5 introduces "venue identity" as a strategic lens, noting the tension between student and corporate positioning

### What did NOT improve (Jack's diagnosis confirmed)

1. **Still 22 agents** — this is hardcoded (`max_agents_allowed = int(num_entities * 0.9)`). Adding $$$ to Zep won't fix it. Only denser seed entities unlock larger populations.
2. **Still no genuine dissent** — Dolans is still "supporting CASK" and "considering collaboration" rather than counter-programming. A 30-year live music venue wouldn't respond this way in reality.
3. **Still no Residence / coexistence stress test** — Sim 1's whole purpose was market validation, but the three-audience dynamic (Residence corporates, students, Nelligan's locals) is still barely present.
4. **Still agreeable tone** — even the caveat from the corporate event booker reads like a polite mention, not a booking walk-away.
5. **Arvin Grewal's "soft, enchanting lights" problem** — the original report flagged this as a hallucination. In the new report he's now described as "planning exciting events" — still disconnected from the car park chaos described in the seed.

### The confirmed root-cause ranking

| Factor | Impact on report quality | Fix |
|---|---|---|
| Seed entity density (22 entities → 22 agents cap) | 🔴🔴🔴🔴🔴 | Rewrite seed with 220+ entities |
| Seed framing as conflict vs proposal | 🔴🔴🔴🔴 | Reframe tensions as unresolved, not resolved |
| Model capability (4o-mini → 4o) | 🟡🟡 | ✅ Done, modest improvement |
| Round count (40 rounds) | 🟡🟡 | Increase to 100 if feasible |
| Report Agent model | 🟡 | ✅ Done (this exercise) |

Conclusion: the Report Agent model upgrade was worth doing (free lift to structure + nuance) but does NOT substitute for the seed rewrite Jack identified. **Seed density + conflict framing is the 80/20 next move.**
