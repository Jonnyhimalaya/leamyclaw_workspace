## C3. UiPath — "Standard agents vs. Advanced agents"
- **URL:** https://www.uipath.com/blog/ai/technical-tuesday-standard-agents-vs-advanced-agents-whats-different-and-why

### Content
UiPath ($7bn RPA giant) draws a public line:

| | Standard / "Agents 1.0" | Advanced / "Agents 2.0" / "Deep agents" |
|---|---|---|
| Loop | ReAct (think→act→observe) | Plan → execute → replan |
| Best for | Bounded tasks (Q&A, summarise, draft) | Long-horizon multi-system workflows |
| Breaks at | Hundreds of steps, context rot | Designed for hours/days |
| Recovery | Weak — often escalates | Replans, switches approach |

**4 pillars of Advanced agents:**
1. **Explicit planning** (markdown to-do, status tracking, replan on failure)
2. **Sub-agent delegation** (specialised, parallel, task-scoped context)
3. **Skills + heavy system prompts** (codified SOPs, compliance, business logic)
4. **Filesystem as memory** (durable scratchpad, resumable, references not full reloads)

**The "4 signals test"** — if any 2 apply, you need an advanced agent:
- Long-horizon with handoffs
- Inspectable evidence trail required
- Parallelism required
- Context that can't fit a prompt

### Why this matters (most of all)
UiPath is publicly endorsing the exact architecture Jonny already ships. Every pillar maps 1:1 to current OpenClaw deployments:

| UiPath pillar | Our equivalent |
|---|---|
| Explicit planning | tasks.md, priority-map.md, daily NDD memory |
| Sub-agent delegation | Multi-agent team (Wrench, Prism, Radar, Pixel, Strategist), `sessions_spawn` |
| Skills + heavy prompts | Skills system, AGENTS.md, SOUL.md, MEMORY.md |
| Filesystem as memory | Workspace + memory-core dreaming + topic files |

This is enterprise-grade air-cover for the consultancy. We can stop having to *invent* the framing — UiPath is selling it for us.

### Actionable
- [ ] **Steal the "4 signals test" verbatim** as a discovery-call qualification tool. Add to consultancy onboarding script.
- [ ] Use **"Standard vs Advanced Agents"** framing on consultancy landing page. Most prospects' mental model = ChatGPT (Standard). We sell something different.
- [ ] Reference UiPath in pitch deck for CFO air-cover ("UiPath, the enterprise RPA standard, calls this Advanced Agents 2.0; we've been deploying it for 6 months")
- [ ] **Audit Kate + Faye MCs against the 4 pillars.** If sub-agent delegation is missing, that's a Phase 2 upsell UiPath now legitimises.
- [ ] Position **Dano's Ledger as the flagship "audit-grade agent system"** case study (matches their inspectable-evidence-trail pillar) for any future finance/audit/regulated-industry pitch
- [ ] Draft blog post: "UiPath says Advanced Agents need 4 things. We deliver all 4 — without UiPath." Targets the exact audience reading their content.

### Strategic note
We can't out-spend UiPath. Win on **deployment speed, openness, price.** Their Advanced Agents = six-figure enterprise contracts; ours = €X00–€X,000/mo with founder-grade attention. That gap is the wedge.
